# Flipkart Product Scraper

A Python Selenium scraper that extracts product data from Flipkart.

## Features
- Scrapes product title, price, image, and link
- Handles pagination
- Saves data to CSV
- Works with dynamic content

## Tech Stack
- Python
- Selenium
- Pandas

## Output
CSV file containing scraped products.
